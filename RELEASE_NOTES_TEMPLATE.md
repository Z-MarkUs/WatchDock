## üöÄ Quick Start

1. **Download** the executable for your platform above
2. **Run** the GUI: `WatchDock.exe` (Windows) or `WatchDock.app` (macOS) or `WatchDock-0.1.0-linux` (Linux)
3. **Configure:**
   - Set up watched folders
   - Add your AI API key (OpenAI, Anthropic, or use Ollama for local)
   - Choose operation mode (Auto or HITL)
4. **Start monitoring:** Run `watchdock` from command line
